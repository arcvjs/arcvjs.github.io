Install database MySQL di linux dan membuat relation antar tables

Update & upgrade server

sudo apt update

lalu

sudo apt upgrade

Install mariadb sql server & client

untuk install sql server

sudo apt install mariadb-server
```bash

untuk install sql client

```bash
sudo apt install mariadb-client

start database server

sudo systemctl start mariadb

Login the database

sudo mysql -u root

gunakan sudo sehingga kita tidak perlu memasukan password

buat database baru

CREATE DATABASE User;

pilih database itu dengan

use User;

Buat templete tabel baru.

CREATE TABLE customers (
    customer_id INT PRIMARY KEY,
    customer_name VARCHAR(50),
    email VARCHAR(100)
);

dan juga

CREATE TABLE orders (
    order_id INT PRIMARY KEY,
    order_date DATE,
    total_amount DECIMAL(10, 2),
    customer_id INT,
    FOREIGN KEY (customer_id) REFERENCES customers(customer_id)
);

dimana orders memiliki foreign key customer id yang merefrensi customer id ( relation).

sekian terimakasih.
